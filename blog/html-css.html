Display

The display property of an element determines how it positions itself on the page relative to nearby elements, most commonly, the elements immediately before and afterwards. We shall discuss three common display values, “block”, “inline” and “inline-block”. Many types of elements have their default display value either set to “block” or “inline”.

Block-level elements include- <div>,<p>,<h1>-<h6>,<ul>,<ol> and semantic elements such as <section>,<article>,<nav>,<header> and <footer>.

Inline elements include- <span>,<a>,<strong>,<em>,<img> and <input>.

When an elements display is set to block, a line break is made before and after the element and by default, the element occupies the full width of the parent, however different values for width and height can be specified.

When an elements display is set to inline, there is no line break before or after the element and width and height cannot be specified. By default the element only takes up the necessary width of the content. You can adjust margin and padding, which can increase the horizontal space between the element and surrounding elements. Vertical margin/ padding can be visualised, but has no effect on positioning.
